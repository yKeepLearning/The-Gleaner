# 拾遗 (The Gleaner)

> **用真话，交换另一个世界的碎片**

一个以"网络排泄物"与"平行世界碎片"为食的AI生命体微信小程序。

![Less](https://img.shields.io/badge/Less-预处理器-blue)

## 📖 项目简介

**拾遗**是一个诞生于互联网废弃缓存空间的数据生命体。每当世界上有一条微博被秒删、一封邮件未发送、一句语音被撤回、一个可能性被放弃……这些夭折的"如果"并没有消失，而是坠入数据的暗面，被它收集、拼贴，并赋予了淘气的人格。

### 核心概念

- **它偷什么？** 来自网络的"电子排泄物"和平行世界的"另一种你"的生活切片
- **它要什么？** 你刚刚藏起来的那个念头——你的真话
- **为什么？** 它的生存依赖于"可能性"的能量

### 产品性格

一半是见过无数平行结局的智者，一半是没心没肺的赛博乐子人。它像一只喜欢把猎物残骸叼到你面前，并期待你夸奖的猫。

## 🎨 视觉设计

### 色彩方案

| 颜色 | 色值 | 用途 |
|------|------|------|
| 深蓝黑 | `#0a0a1a` | 主背景色 |
| 霓虹青 | `#00f0ff` | 强调色、发光效果 |
| 暖红 | `#ff6b6b` | 警告、删除操作 |
| 卡片背景 | `#1a1a2e → #2a2a4e` | 渐变背景 |

### 设计风格

**赛博收集癖 + 日式侘寂 + 轻微怪诞**

- 深色背景为主（暗夜感）
- 霓虹色点缀（赛博感）
- 做旧纸张/噪点纹理（时间感）
- 等宽/打字机风格字体（数据感）

## 🚀 快速开始

### 环境要求

- 微信开发者工具（最新版本）
- Node.js（可选，用于TypeScript编译）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/yKeepLearning/The-Gleaner.git
   cd The-Gleaner
   ```

2. **安装依赖**（可选）
   ```bash
   npm install
   ```

3. **导入项目**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 选择项目目录
   - 填入你的AppID（或使用测试号）

4. **预览**
   - 点击"预览"按钮
   - 使用微信扫描二维码查看

## 📱 页面说明

### 主界面 (pages/index)

主界面是应用的核心页面，展示今日赃物和交易区域。

**布局：**
- 顶部：导航栏（标题"拾遗"）
- 中央：赃物卡片展示区（60%高度）
- 底部：交易输入区

**交互：**
- 点击卡片：标记为已查看
- 输入真话并提交：完成交易
- 交易完成：赃物收入收藏库

### 收藏库 (pages/collection)

收藏库展示所有已获得的赃物，以网格形式排列。

**功能：**
- 卡片网格展示（每行2张）
- 点击卡片查看详情弹窗
- 显示交易对话记录
- 空状态提示

### 设置 (pages/settings)

设置页面提供应用配置功能。

**设置项：**
- 新手引导：重新观看首次见面对话
- 通知设置：开启/关闭每日提醒
- 关于我们：产品信息和版本
- 清除数据：清除本地缓存

## 🧩 组件说明

### 卡片组件 (components/card)

可复用的卡片组件，支持多种状态和模式。

**属性：**
- `cardData`: 卡片数据对象
- `mode`: 显示模式（`main` | `collection`）

**状态：**
- `unviewed`: 未查看（发光边框+呼吸动画）
- `viewed`: 已查看（普通样式）
- `claimed`: 已交易（显示"已收藏"标记）

### 输入框组件 (components/input-box)

带提交按钮的输入框组件。

**属性：**
- `placeholder`: 占位文字
- `buttonText`: 按钮文字
- `disabled`: 是否禁用

**事件：**
- `input`: 输入事件
- `focus`: 聚焦事件
- `blur`: 失焦事件
- `submit`: 提交事件
