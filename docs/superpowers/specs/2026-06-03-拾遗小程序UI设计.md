# 拾遗小程序UI设计文档

> **项目名称**：拾遗（The Gleaner）
> **设计范围**：仅UI界面（前端页面，不涉及后端和AI服务）
> **设计日期**：2026-06-03

---

## 一、整体架构

### 1.1 页面结构

- **3个页面**：
  - `pages/index`（主界面）
  - `pages/collection`（收藏库）
  - `pages/settings`（设置）

- **导航方式**：底部Tab导航
  - 图标风格：线条简约 + 霓虹点缀
  - Tab项：主界面（🏠）、收藏库（📦）、设置（⚙️）

### 1.2 技术栈

- **框架**：微信小程序原生开发
- **语言**：TypeScript + Less
- **组件框架**：glass-easel（已配置）

### 1.3 全局色彩方案

- **主背景色**：#0a0a1a（深蓝黑色）
- **霓虹青色**：#00f0ff（用于强调、发光效果）
- **暖红色**：#ff6b6b（用于警告、删除等操作）
- **卡片背景**：渐变（#1a1a2e → #2a2a4e）
- **文字颜色**：#ffffff（主文字），#888888（次要文字）

---

## 二、主界面设计（pages/index）

### 2.1 页面布局

- **顶部**：状态栏 + 标题"拾遗"
- **中央**：赃物卡片展示区（占屏幕60%高度）
- **底部**：交易输入区（输入框 + 提交按钮）

### 2.2 赃物卡片设计

- **尺寸**：宽300rpx，高400rpx
- **样式**：
  - 圆角：16rpx
  - 阴影：轻微阴影效果
  - 背景：渐变（深蓝→深紫）
  - 边框：1rpx solid rgba(0, 240, 255, 0.3)

- **卡片内容**：
  - 中央：赃物类型图标（文字/图片/音频）
  - 底部：获得日期（格式：YYYY-MM-DD）

- **卡片状态**：
  - **未查看**：发光边框（霓虹青色），轻微呼吸动画
  - **已查看**：普通样式，无动画
  - **已交易**：卡片右下角显示"已收藏"标记

### 2.3 交易输入区设计

- **输入框**：
  - 圆角：8rpx
  - 背景：半透明深色（rgba(255, 255, 255, 0.1)）
  - placeholder文字："用真话交换..."
  - 高度：80rpx

- **提交按钮**：
  - 霓虹色渐变背景（青色→蓝色）
  - 文字："交换"
  - 圆角：8rpx
  - 高度：80rpx

- **交易请求文字**：
  - 位于输入框上方
  - 字体：等宽/打字机风格
  - 带闪烁光标效果

### 2.4 主界面背景

- **主背景**：深色渐变（#0a0a1a → #1a1a2e）
- **纹理**：轻微噪点纹理（增加质感）
- **可选效果**：微弱的霓虹光晕效果

---

## 三、收藏库设计（pages/collection）

### 3.1 页面布局

- **顶部**：
  - 标题"可能性博物馆"
  - 可选：筛选按钮（按类型/日期筛选）

- **内容区**：瀑布流/网格布局
  - 每行2张卡片
  - 可滚动浏览
  - 支持下拉刷新

### 3.2 卡片设计

- **尺寸**：宽200rpx，高260rpx
- **视觉纹理**（根据赃物类型不同）：
  - **类型A（电子排泄物）**：揉皱纸张质感，暖色调
  - **类型B（平行世界切片）**：拍立得照片质感，冷色调
  - **类型C（稀有掉落）**：烧焦信封质感，霓虹色点缀

- **卡片信息**：
  - 右上角：获得日期
  - 中央：赃物缩略图/文字预览

### 3.3 交互细节

- **点击卡片**：弹出详情弹窗
  - 显示完整内容
  - 显示交易对话记录
  - 关闭按钮

- **空状态**：
  - 显示文字："还没有收藏任何赃物"
  - 副文字："每天来这里看看它带回了什么"

## 四、设置页面设计（pages/settings）

### 4.1 页面布局

- **列表式布局**：每个设置项一行
- **每行结构**：图标 + 标题 + 右侧箭头/开关

### 4.2 设置项

1. **新手引导**
   - 图标：📖
   - 功能：重新观看首次见面对话
   - 右侧：箭头（跳转）

2. **通知设置**
   - 图标：🔔
   - 功能：开启/关闭每日赃物到达提醒
   - 右侧：开关控件

3. **关于我们**
   - 图标：ℹ️
   - 功能：产品介绍、版本信息
   - 右侧：箭头（跳转）

4. **清除数据**
   - 图标：🗑️
   - 功能：清除本地缓存
   - 右侧：箭头（跳转）
   - 文字颜色：暖红色（警告）

### 4.3 视觉风格

- **设置项背景**：半透明深色卡片
- **图标风格**：线条简约，霓虹色点缀
- **分隔线**：细线，低对比度（rgba(255, 255, 255, 0.1)）

### 4.4 关于页面内容

- **产品名称**：拾遗
- **副标题**：用真话，交换另一个世界的碎片
- **版本号**：v1.0.0
- **简短介绍**：一个诞生于互联网废弃缓存空间的数据生命体

## 五、组件设计

### 5.1 全局组件

- **导航栏组件**（已存在）：`components/navigation-bar`
- **卡片组件**：`components/card`（可复用于主界面和收藏库）
- **输入框组件**：`components/input-box`

### 5.2 动画效果

- **呼吸动画**：未查看的赃物卡片轻微缩放（scale 1.0 → 1.02）
- **发光效果**：霓虹色边框发光（box-shadow）
- **打字机效果**：交易请求文字逐字显示
- **页面切换**：底部Tab切换时的淡入淡出效果

## 七、注意事项

1. **视觉清晰度**：界面要清晰，不要让人看不清楚
2. **性能优化**：卡片数量多时注意性能（虚拟列表/懒加载）
3. **响应式设计**：适配不同屏幕尺寸
4. **无障碍设计**：考虑色盲用户，不仅依赖颜色区分状态
